Java framework for modern web, cloud, high-load, and microservices solutions

A full-featured, modern Java platform built from the ground up as an alternative to Spring/Micronauts/Netty/Jetty. It is designed to be self-sufficient (no third-party dependencies), simple, lightweight and provides competitive performance. It consists of a number of libraries, from dependency injection and high-performance asynchronous I/O (inspired by Node.js), to application servers and big data solutions

Minimum dependencies

Lightweight minimalistic self-sufficient Java framework without third-party dependencies

Rich stack

Features a rich stack of technologies for building fast asynchronous applications

Overhead-free

Minimalist design with no overhead for competitive performance

Alternative approach

An alternative to Spring, Micronaut, Apache Spark, Quarkus, and other solutions

Components

A simple set of loosely coupled modules without third-party dependencies

Async-IO

Asynchronous IO processing

High-performance asynchronous IO with efficient event loop, promises and streaming protocols. Inspired by the Node.js approach

HTTP

One of the fastest HTTP servers

Asynchronous HTTP server and client with WebSocket support. Suitable for handling millions of requests per second

Inject

Dependency injection library

Optimized for fast application startup and runtime performance. Provides dependency graph introspection capabilities

Serializer

The fastest JVM serializer

Annotation-based serialization with dynamically generated serializers. Support for subclasses, collections, and specialized collections

Codegen

Class generation on the fly

Dynamic bytecode generation on top of ObjectWeb ASM library. Create custom classes on the fly using Lisp-like AST expressions

Specializer

Speeding up your code

Improve class performance at runtime by automatically rewriting classes, replacing dynamic innovations with static method calls

FS

Managing file systems

Tiny asynchronous abstraction on top of common file operations. Local or remote file storages, as well as cluster storages for big data

RPC

Binary protocol for microservices

Library for high-load distributed applications. An alternative to microservice architecture without HTTP overhead
